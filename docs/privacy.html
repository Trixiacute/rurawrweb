1. Introduction

This privacy policy explains how we collect, use, and protect your personal information when you use our Discord bot. By using the bot, you consent to the data practices described in this policy.

2. Information We Collect

2.1 User Information:

  • Discord User ID
  • Last.fm username (when provided)
  • Command preferences
  • Feature usage patterns

2.2 Server Information:

  • Server ID
  • Channel IDs for specific features
  • Server prefix settings
  • Feature configurations

3. Purpose of Data Collection

We collect this information to:

  • Connect your Discord account with your Last.fm profile
  • Provide personalized music recommendations and statistics
  • Remember your preferences and settings
  • Improve the bot based on usage patterns
  • Ensure proper functioning of all features

4. Data Deletion

You can request the deletion of your data at any time by using the following methods:

  • Using the !datadelete command in any server where the bot is present
  • Contacting us directly via our support server

When you request data deletion, we will remove all personal information associated with your Discord account from our database.
